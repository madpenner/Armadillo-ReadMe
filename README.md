# Armadillo-ReadMe
# Hello Armadillo Jan – Apr 2022

• Created a scene with an armadillo and implemented features to allow repositioning and shape shifting, shading, and shadow mapping, using THREE js and GLSL.

• Used Javascript to implement keyboard commands allowing an armadillo object to move and rotate in a scene, created an orb to use as a light source for the armadillo to reflect, caused the orb to oscillate sinusoidally over time, and implemented lasers to point from the armadillo’s eyes to the moving orb, within a distance threshold.


• Utilized knowledge of lighting and shading to modify the appearance of 3D models. Used basic textures and implemented various shading algorithms such as Blinn-Phong, Toon, and a rolling grid pattern.


• Implemented a skybox using cube mapping and used environment mapping to make the armadillo reflect the scene in the skybox. Implemented shadow mapping using multipass rendering and used multisampling to give the armadillo a realistic and non-jagged shadow based off a moving light source in the environment.
